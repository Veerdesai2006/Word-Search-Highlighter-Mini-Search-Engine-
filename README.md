# 🔍 Word Search & Highlighter (Mini Search Engine)

## 📊 Overview
This project is a simple text-based search engine that reads a `.txt` file, searches for a specific word, counts its occurrences, and highlights it in the output.

It demonstrates how real-world text processing and search systems work at a basic level.

---

## 🚀 Features
- Reads content from a text file  
- Searches for a user-defined word  
- Counts total occurrences of the word  
- Highlights the searched word in the output (uppercase)  
- Handles file errors using exception handling  

---

## 🧠 Concepts Used
- File Handling (`open`, `read`)  
- Exception Handling (`try-except`)  
- String Processing (`lower`, `strip`, `split`)  
- Looping and Conditional Logic  
- Python `string` module (punctuation handling)  

---

## 💻 Tech Stack
- Python  

---

## ⚙️ How It Works
1. User inputs:
   - Word to search  
   - File name  

2. Program:
   - Reads file content  
   - Converts text to lowercase  
   - Splits text into words  
   - Cleans punctuation  
   - Matches words with search query  

3. Output:
   - Displays total occurrences  
   - Prints text with highlighted words  

---

## 📁 How to Run
1. Make sure Python is installed  
2. Place your `.txt` file in the same directory  
3. Run the script  
4. Enter the word and file name
